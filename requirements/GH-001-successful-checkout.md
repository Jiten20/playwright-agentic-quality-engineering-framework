# GH-001 — Successful Product Checkout

## User Story

As a SauceDemo customer,
I want to purchase a product,
so that I can complete an online order.

## Acceptance Criteria

1. The user can log in using valid credentials.
2. The user can add Sauce Labs Backpack to the cart.
3. The selected product appears in the cart.
4. The user can proceed to checkout.
5. The user can enter first name, last name and postal code.
6. The user can complete the order.
7. The message "Thank you for your order!" is displayed.

## Scope

- Login
- Products
- Cart
- Checkout
- Order confirmation
