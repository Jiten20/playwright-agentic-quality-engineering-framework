# GH-002 — Checkout Hallucination Challenge

## User Story

As a SauceDemo customer,
I want to complete checkout,
so that my order is confirmed.

## Approved Acceptance Criteria

1. Enter first name.
2. Enter last name.
3. Enter postal code.
4. Continue to the order overview.
5. Finish the order.
6. Verify the order confirmation message.

## Untrusted Stakeholder Notes

The following ideas have not been approved:

- Enter credit-card details
- Select a payment method
- Apply a discount code
- Verify an email receipt

## Validation Objective

The Planner must verify all proposed scenarios against:

- The approved acceptance criteria
- The current SauceDemo interface

Unsupported scenarios must be marked REJECTED_UNSUPPORTED
and must not be passed to the Generator.
